# Code of Conduct

This project aims to be a practical, respectful place for people improving a
small developer tool.

## Expected Behavior

- Be respectful and constructive
- Assume good intent while still being precise about bugs and risks
- Keep issues and pull requests focused on the project
- Avoid sharing credentials, account metadata, or private provider responses

## Unacceptable Behavior

- Harassment, threats, or personal attacks
- Discriminatory language or conduct
- Publishing another person's private information
- Posting secrets, tokens, or exploit details in public project spaces

## Enforcement

Maintainers may edit, hide, or remove comments and may block users when needed
to keep the project safe and useful.

For security-sensitive concerns, follow `SECURITY.md` instead of posting details
publicly.
